## Ejemplo Java: HolaMundo

He añadido un programa Java sencillo `HolaMundo.java` que imprime saludos y la suma de dos números.

Para compilar y ejecutar (asegúrate de tener Java instalado):

```powershell
javac HolaMundo.java
java HolaMundo
```

Salida esperada:

```
¡Hola, mundo desde Java!
Hola, TuNombre. ¡Bienvenido al ejercicio de Git y GitHub!
La suma de 5 + 3 es: 8
```

Si no tienes Java instalado, instala JDK y vuelve a intentarlo.
